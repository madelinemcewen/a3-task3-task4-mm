Summary table with mean, median, sd, and n

  • Use group_by() and summarize()
urchin_stats <- urchin_data %>% 
  group_by(common_name) %>% 
  summarize(mean = mean(size), 
            standard_deviation = sd(size), 
            median= median(size), 
            n=n())

Looking at the histogram ggplots, the red urchin is much more evenly spread than the purple. Red also has a smaller standard deviation which indicates the values are closer to the mean, which we can see when we look at the results from our graphs and summary table. While the means between species differ by only about 2.3, the range of sizes is only between 2.5 and 15. Keeping this in mind, a difference of 2.3 is actually quite large.

Exploring purple urchins

  • Use mutate() and case_when()
purple_urchins<-urchin_data %>% 
  filter(common_name == "Purple Urchin") %>% 
  mutate(Date=ymd(date)) %>% 
  mutate(year= year(Date)) %>% 
  mutate(site_full=case_when(
    site == "NAPL" ~"Naples Reef",
    site == "AQUE" ~"Arroyo Quemado Reef",
    site == "CARP" ~"Carpinteria Reef",
    site == "MOHK" ~"Mohawk Reef",
    site == "IVEE" ~ "Isla Vista Reef"
  ))
